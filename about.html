About This Project

Oops Mart is a front-end practice project created by Amanat Ali. This website was developed after completing the Introduction to Front-End Developer certificate from Meta.

During this learning journey, I gained practical knowledge of HTML, CSS, and Bootstrap. To apply these skills in a real project, I designed and developed Oops Mart.

Oops Mart contains four demo stores:

  • Electronic Store
  • Clothes Store
  • Shoes Store
  • Kitchenware Store

This website is created only for learning, practice, and portfolio purposes. It is not a real online shopping website. No products are sold, and no online purchasing or payment system is available.
